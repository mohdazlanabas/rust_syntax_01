# Rust Tutorial

A Rust language practice project for learning and exploring Rust syntax and features.

## Project Structure

```
rust_syntax_01/
├── src/
│   ├── main.rs          # Main entry point
│   ├── decode.rs        # Decode module
│   ├── main_part01.rs   # Part 1 exercises
│   ├── main_part02.rs   # Part 2 exercises
│   ├── main_part03.rs   # Part 3 exercises
│   ├── main_part04.rs   # Part 4 exercises
│   └── main_part05.rs   # Part 5 exercises
├── Cargo.toml
└── README.md
```

## Dependencies

- `rand` (0.8.5) - Random number generation
- `regex` (0.2) - Regular expressions
- `reqwest` (0.11.11) - HTTP client

## Building and Running

Build the project:
```bash
cargo build
```

Run the project:
```bash
cargo run
```
